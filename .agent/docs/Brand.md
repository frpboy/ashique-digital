## 1. Brand Essence

### 1.1 Mission

To help ambitious businesses build systemized, predictable growth — not just run ads.

### 1.2 Vision

To be the most trusted growth strategy partner for Indian SMEs and startups who are serious about scaling.

### 1.3 Brand Personality

| Trait         | What it Looks Like in Practice                                 |
| ------------- | -------------------------------------------------------------- |
| **Confident** | States opinions clearly. Doesn't hedge or overqualify.         |
| **Direct**    | Gets to the point. Leads with outcomes, not process details.   |
| **Warm**      | Approachable. Uses plain language. Avoids buzzwords.           |
| **Strategic** | Backs claims with data. Thinks systems, not tactics.           |
| **Ambitious** | Talks about growth like it's inevitable with the right system. |

### 1.4 Brand Voice Attributes

| Attribute  | ✅ Do                             | ❌ Don't                              |
| ---------- | --------------------------------- | ------------------------------------- |
| Tone       | Conversational yet professional   | Corporate jargon or salesy            |
| Confidence | "Here's what works."              | "We think perhaps this might work..." |
| Proof      | Lead with specific metrics        | Vague claims like "results-driven"    |
| Empathy    | Mirror the client's pain first    | Jump straight to the solution pitch   |
| Language   | Plain English (or Indian English) | Marketing clichés like "synergy"      |

## 2. Logo Usage

### 2.1 Primary Wordmark

```
Ashique · Digital
```

- **Font:** Syne Bold (700)
- **Color (light backgrounds):** `#0D1B2A`
- **Color (dark backgrounds):** `#FFFFFF`
- **Dot separator:** Electric Teal `#00C2CB`
- **Size:** Never below 120px wide in digital

### 2.2 Logo Clear Space

Maintain clear space equal to the height of the letter "A" on all sides of the logo.

### 2.3 Logo Don'ts

- ❌ Do not change the font
- ❌ Do not use purple, red, or pink colors for the logo
- ❌ Do not add drop shadows or gradients to the wordmark
- ❌ Do not stretch or compress the logo
- ❌ Do not place on busy backgrounds without a solid backing

## 4. Typography

### 4.1 Primary Fonts

| Role       | Font             | Weight  | Use Case                             |
| ---------- | ---------------- | ------- | ------------------------------------ |
| Display/H1 | Syne             | 800     | Hero headlines, major section titles |
| Headings   | Syne             | 700     | H2, H3 section titles                |
| Body       | Inter            | 400–500 | All body copy, paragraphs, UI        |
| Accent     | Playfair Display | 400–700 | Pull quotes, testimonials, special   |

### 4.2 Font Usage Rules

- **Syne** is reserved for headings only — never use for long body copy
- **Inter** is the workhorse — clear, readable at all sizes
- **Playfair Display** is used sparingly — testimonial quotes, special callouts only
- **Minimum body text size:** 16px for readability

### 4.3 Heading Hierarchy

```
H1 — Syne 800 — 80px / 2.5rem mobile
H2 — Syne 700 — 48px / 2rem mobile
H3 — Syne 700 — 24px / 1.25rem mobile
Body — Inter 400 — 18px / 16px mobile
```

## 5. Tone of Voice Guide

### 5.1 Copy Framework: AIDA for Ashique

| Stage     | Goal                     | Example                                     |
| --------- | ------------------------ | ------------------------------------------- |
| Attention | Interrupt the scroll     | "Stop running ads without a system."        |
| Interest  | Mirror the pain          | "Most SMEs waste 70% of their ad budget."   |
| Desire    | Show the transformation  | "What if leads came in on autopilot?"       |
| Action    | Clear, friction-free CTA | "→ Book your free 30-minute strategy call." |

### 5.2 Writing Rules

1. **Lead with outcomes**, not process: "Get 3x more qualified leads" not "We use proprietary methods"
2. **Use specific numbers**: "220% increase" > "significant growth"
3. **Short paragraphs**: Max 3 sentences per paragraph on web
4. **Active voice**: "We build funnels" not "Funnels are built"
5. **Conversational punctuation**: Em dashes, ellipses where natural
6. **No corporate clichés**: Never use "leverage", "synergy", "paradigm shift"

### 5.3 Sample Copy Reference

**✅ Strong (Ashique's voice):**

> "Most businesses don't have a marketing problem. They have a systems problem. I fix that."

**❌ Weak (generic agency speak):**

> "We leverage data-driven strategies to optimize your marketing ROI and deliver measurable outcomes."

## 6. Photography Direction

### 6.1 Ashique's Photos

| Context           | Style                                                    |
| ----------------- | -------------------------------------------------------- |
| Hero photo        | Professional, direct gaze, clean background or workspace |
| About page        | Candid, working — laptop, whiteboard, strategy session   |
| Case study author | Headshot, warm lighting, confident posture               |

### 6.2 Rules

- **Always use Ashique's real photos** — no AI-generated or stock people photos
- **Consistent color grade** — slightly warm-toned, professional
- **Natural settings** — avoid sterile studio shots; prefer real workspaces
- **No heavy filters** — keep it authentic, not Instagram-edited

### 6.3 Supplementary Image Style

- Abstract geometric shapes in navy/teal for section backgrounds
- Data visualization graphics (charts, funnels, arrows) as concept illustrations
- Minimal, clean product mockups for case study visuals

## 7. Social Media Guidelines

| Element       | Guideline                                                  |

| Profile photo | Professional headshot with clean background                |
